# Nexus – Investor & Entrepreneur Collaboration Platform

## Project Structure
- `src/components/` – Reusable UI components
- `src/pages/` – All page-level components
- `src/context/` – Global state management
- `src/data/` – Mock/static data
- `src/types/` – TypeScript interfaces

## Tech Stack
- React 18 + TypeScript
- Tailwind CSS
- React Router DOM
- Axios
- Lucide React (icons)

## Run Locally
npm install
npm run dev

## UI Theme
- Primary: Blue (#3B82F6)
- Secondary: Teal (#14B8A6)  
- Accent: Amber (#F59E0B)
- Font: Inter var
- Animations: fade-in, slide-in
