## 📋 Project Structure
- `src/components/` – Reusable UI components
- `src/pages/` – All page-level components
- `src/context/` – Global state management
- `src/data/` – Mock/static data
- `src/types/` – TypeScript interfaces

## 🛠️ Tech Stack
- React 18 + TypeScript
- Tailwind CSS
- React Router DOM
- FullCalendar
- Lucide React (icons)
- Axios
- React Signature Canvas
